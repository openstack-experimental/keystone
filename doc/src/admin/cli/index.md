# Command-line tools

Keystone-NG provides two command-line programs:

- [`keystone`](keystone.md) runs the API service or emits its OpenAPI schema.
- [`keystone-manage`](keystone-manage.md) performs bootstrap, catalog, database,
  key, OAuth2, and distributed-storage administration.

Run either command with `--help`, and run nested `keystone-manage` commands with
`--help`, for the exact options in the installed version.
