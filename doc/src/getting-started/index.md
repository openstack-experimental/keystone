# General Information

Keystone-NG is a Rust implementation of OpenStack Identity capabilities. It can
run beside Python Keystone, with a reverse proxy routing requests to the
implementation that owns each endpoint.

Use this section for information that applies to operators, API consumers, and
contributors:

- [Architecture](architecture.md) describes the service and its dependencies.
- [Compatibility](compatibility.md) explains the relationship with Python
  Keystone and the v3 and v4 APIs.
- [Performance comparison](performance.md) explains how performance results are
  produced and where to find current measurements.

For task-oriented documentation, continue with the [installation guide](../install/index.md),
[user documentation](../user/index.md), or
[administrator guides](../admin/index.md).
