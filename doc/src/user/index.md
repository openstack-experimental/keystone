# User Documentation

This section is for people and applications consuming Keystone-NG APIs,
including delegated domain managers.

- [API guide](api.md) covers version discovery, OpenAPI, and common resource
  requests.
- [Authentication](authentication/index.md) covers password, TOTP, passkeys,
  and token headers.
- Feature guides describe the client side of Fernet tokens, federation,
  OAuth2/OIDC, SCIM and its API keys, Kubernetes authentication, and identity
  mapping.

Each feature guide links to its administrator counterpart when deployment or
configuration work is required.
